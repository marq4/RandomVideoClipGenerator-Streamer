# Random Video Clip Generator / Streamer
Consolidated monorepo for Python core, Docker generator, and AWS Cloud service.

Play random sections of music videos (for background visual noise).
Great for parties / DJs!

### Common requisites: ###
* The music videos have to be locally available.

# Python Core (for Windows)
Visit [the website](https://www.randomvideoclipgenerator.com), download the Python script, run it.
You will need:
* VLC (Free, open-source video player for Windows & Linux).
* Python.
* FFmpeg.

# Random Video Clip Streamer - Docker
Run the Docker container, watch the random video clip stream directly in your web browser (such as FireFox)!

You will need:
* Docker Desktop (available for Windows and Linux).

For instructions please visit the [the website](https://www.randomvideoclipgenerator.com).

# Random Video Clip Generator Cloud service
Visit the [the website](https://www.randomvideoclipgenerator.com), download the script (Bash or PowerShell), generate a list (text file) of your music videos, upload it: a playlist (xml) will be downloaded by your browser. Double-click it to open up the random video clips on VLC!
